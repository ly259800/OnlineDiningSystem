# OnlineDiningSystem
一、运行环境：  
开发工具为eclipse  
java环境为jdk1.8  
使用的服务器为tomcat1.8  
数据库为mysql5.5  
二、需求分析：  
功能需求如下  
a.用户可以选择空闲的餐桌并查看菜单  
b.用户可以根据食品分类查看该类食物  
c.用户可以选择喜欢的食物并添加到购物车  
d.当用户选择结账时，会提醒管理员进行结账  
e.管理员可以对餐桌，食品类型，和食物进行增删改操作，还有对订单进行管理  
三、实现方式：  
本项目使用SSM框架整合实现开发  
1.使用Mybatis框架实现数据与数据库的交互，即实现对数据库的管理;  
2.使用Spring实现服务层，即业务层，使得控制层和数据访问层可以进行交互。  
3.使用SpringMVC实现表现层，即控制层，通过调用服务层来实现静态页面的动态化。
